# 실제 화면 캡처 넣는 곳

아래 파일들을 이 `assets` 폴더에 그대로 넣어주세요. (PNG 권장, 데스크톱 넓은 화면에서 캡처)
넣은 뒤 파일명을 알려주시면, 각 캡처 위에 광고 노출 위치를 표시해 페이지에 반영합니다.

## 필요한 캡처 (★ = 우선순위 높음)

| 파일명 | 캡처할 화면 | 쓰이는 곳 |
|---|---|---|
| ★ `main.png` | 요즘IT **메인(랜딩) 화면** — 콘텐츠 피드 + 우측 사이드바가 보이게 | 콘텐츠·컬렉션·카테고리 탭, 배너 탭(랜딩 LD-1·LD-2) |
| ★ `article.png` | **기사(콘텐츠) 상세 페이지** — 우측 사이드바(CD-1·CD-2) + 본문 하단(CD-3) 영역이 보이게 | 배너 탭(콘텐츠 페이지 CD-1·2·3) |
| ★ `newsletter.png` | **Pick-IT 뉴스레터** — 하단 배너 영역이 보이게 | 뉴스레터 탭(Pick-IT 배너) |
| `x-post.png` | **X(트위터) 게시물** 예시 | SNS 탭 |

※ 단독 eDM 은 캡처 대신 실제 발송 링크(Stibee)로 연결 완료 — 편집 구역의 `EDM_EXAMPLES` 참고.

## 팁
- 광고가 실제로 들어간 화면이면 더 좋고, 없으면 빈 화면도 괜찮습니다(위치만 표시).
- 세로로 긴 화면은 광고 영역이 포함된 위쪽 부분만 잘라도 됩니다.
- 파일명은 위와 다르게 해도 되며, 넣은 뒤 알려주시면 맞춰서 연결합니다.
