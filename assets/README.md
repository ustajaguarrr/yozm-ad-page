# 실제 화면 캡처 (assets)

각 탭의 "실제 이렇게 노출됩니다" 미리보기는 여기 이미지들을 사용합니다.
배너 위치는 `index.html`의 `BANNERS` 객체에서 `shot`(캡처 파일)·`iw/ih`(원본 크기)·`rect`(배너 좌표)로 지정합니다.

## 현재 사용 중인 파일

| 파일명 | 화면 | 쓰이는 곳 |
|---|---|---|
| `main.png` | 요즘IT 메인(랜딩) 화면 상단 | 콘텐츠 탭(브랜디드=히어로 배너 위치) |
| `article.png` | 기사 상세 상단 (우측 CD-1·CD-2) | 배너 탭 CD-1 |
| `article2.png` | 기사 상세 하단 (CD-2·CD-3) | 배너 탭 CD-2·CD-3 |
| `ld1.png` | 랜딩 페이지 피드 하단 (위시켓 배너) | 배너 탭 LD-1 |
| `ld2.png` | 랜딩 페이지 하단 (요즘IT 브랜디드 배너) | 배너 탭 LD-2 |
| `category.png` | 메인 화면 + '위시켓' 카테고리 삽입(편집본) | 카테고리 탭 |
| `newsletter.png` | Pick-IT 뉴스레터 | 뉴스레터 탭 |
| `x-post.png` | 요즘IT X(트위터) 게시물 | SNS 탭 |

- `category.png` 는 `main.png`의 카테고리 줄에 '위시켓'을 넣어 만든 **편집본**입니다.
- 단독 eDM·컬렉션은 캡처 대신 실제 링크로 연결 — 편집 구역의 `EDM_EXAMPLES` / `COLLECTIONS` 참고.

## 캡처를 새로 교체할 때
1. 같은 이름으로 이 폴더에 덮어쓰기
2. 화면 크기(가로·세로 픽셀)가 달라졌으면 `index.html`의 해당 배너 `iw/ih` 와 `rect`(배너 위치) 숫자를 맞춰주세요
3. 커밋 → 푸시하면 자동 배포
